## Anaconda for Linux

Please refer to `https://www.anaconda.com/`.

```
$ cd /tmp
$ curl -O https://repo.anaconda.com/archive/Anaconda3-2019.10-Linux-x86_64.sh
$ bash Anaconda3-2019.10-Linux-x86_64.sh
```

add `export PATH=~/anaconda3/bin:$PATH` to `~/.bashrc`
